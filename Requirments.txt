Requirements
    Functional
        1-	User can put any URL.
        2-	User can define level of digging the web (minimum 3)
        3-	User can define the text format (font and size)
        4-	System shall fetch data automatically from the web according to the number of levels which is provided by user.
        5-	The first web page on given URL is level 1.
        6-	System shall convert web pages to a pdf file and create external link for all hyperlinks in the page.
        7-	User can save the produced pdf file with any name.
        
    Nonfunctional:
        1-	Runtime complexity
        2-	Memory complexity
        3-	Language and libraries
